My97DatePicker插件简易使用教程

简介：My97DatePicker提供了一个时间选择框的样式，并且可以设定时间的开始结束、时间的格式、时间级联。

首先引入该包seajs.use(['$', 'My97DatePicker'], function($) {});
实例 源码 说明 
<input type="text" id="input1" onFocus="WdatePicker()"/>
           
使用My97DatePicker插件要先引用该工具包，在input框加上属性
onFocus="WdatePicker()"，就可以显示时间框
            
onFocus="WdatePicker({dateFmt:'yyyy年MM月dd日 HH:mm:ss'})"
            
dateFmt属性定义时间的显示格式，可以随意搭配：
yyyy-MM-dd HH:mm:ss	    2008-03-12 19:20:00
yy年M月	                    08年3月
yyyyMMdd	            20080312
今天是:yyyy年M年d HH时mm分	    今天是:2008年3月12日 19时20分
H:m:s	                    19:20:0
y年	                    8年
MMMM d, yyyy	            三月 12, 2008
            
onfocus="WdatePicker({dateFmt:'yyyy年MM月'})"
            
这里将格式换成dateFmt:'yyyy年MM月'，相应的显示界面也会改变
            
onFocus="WdatePicker({minDate:'2000-12-31 23:59:59',
    dateFmt:'yyyy年MM月dd日 HH:mm:ss',
    maxDate:'2039-12-31 23:59:59'})"
            
minDate:可选最早时间,maxDate:可选最晚时间。
起始： 结束： 
<input type="text" class="Wdate" id="s1" onclick="WdatePicker({
        dateFmt:'yyyy-MM-dd',
        maxDate:'#F{$dp.$D(\'e1\', {d:-1}) || (\'2020-10-01\')}'})"/>
<input type="text" class="Wdate" id="e1" onclick="WdatePicker({
        dateFmt:'yyyy-MM-dd',
        minDate:'#F{$dp.$D(\'s1\', {d:+1}) || (\'2000-01-01\')}'})"/>
            
时间级联:起始时间的最大值设定为结束时间减一天，当结束时间为空时，
则使用设定的最大时间。$dp.$ 相当于 document.getElementById
函数。#F{$dp.$D(\'e1\')||\'2020-10-01\'}表示当e1（id值）
为空时, 采用 2020-10-01 的值作为最大值
