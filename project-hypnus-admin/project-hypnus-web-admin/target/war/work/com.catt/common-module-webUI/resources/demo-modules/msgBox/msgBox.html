msgBox插件简易使用教程

简介：msgBox提供了消息的各种提示方法以及弹出关闭时的数据传送

身份证号: 
首先引入该包seajs.use(['$', 'msgBox'], function($, msgBox){});
样例 源码 说明 
定义： tips: function (msg, time, callback) {***};

调用：msgBox.tips("测试tips");
 
参数含义：

msg：提示内容

time：显示秒数，整数类型（单位：秒）

callback：回调函数
 
定义：warn: function (msg, time, callback) {***};

调用：msgBox.warn("正在测试warn", 1, function(){});
 
参数含义：

msg：提示内容

time：显示秒数，整数类型（单位：秒）

callback：回调函数
 
定义：alert: function (options) {***}；

调用：

msgBox.alert({

    title : 'alert测试',

    msg : 'alert测试内容',

    callback : function() {}

});

                
参数options含义：

支持字符串类型和对象类型 对象类型格式：

{

    title : '这是标题',// 可选

    msg : '这是内容',// 必须

    showCloseBtn:true,

    callback : function() {}// 这是回调函数，可选

}
 
定义：open: function (options) {***};

调用：

msgBox.exWindow.open({

    width : "600px",

    height : "800px",

    url : url,

    close : function(result){

        if(result){

            alert("name:" + result.name + "\n" + "idNum:" + result.idNum);

        }

    }

});

                    
参数options含义：

{

    title:'窗口标题',

    url:'http地址',

    width:'宽度px',

    height:'高度px',

    success:'成功打开时的回调函数,会回传当前的窗口标识ID',

    close:'窗口关闭的时候，回调函数'

}
 
定义：confirm: function (options) {***};

调用：

msgBox.confirm({

    title: '提示',

    msg: '请点击按钮',

    callback: function (btnType) {

        if (btnType == 'ok') {

            alert("刚刚点击了确定按钮");

        }

        if (btnType == 'ok') {

            alert("刚刚点击了取消按钮");

        }

    }

});
 
参数options含义：

{

    title : '这是标题',// 可选

    msg : '这是内容',// 必须

    // 这是回调函数，可选，btnType为点击按钮类型
    callback : function(btnType) {}
}
