util插件简易使用教程

简介：util提供了对字符串、时间、数字、对象、表单、URL的一系列工具

首先引入该包seajs.use(['$', 'util'], function($, util){});

$('#getUrlParam').on('click', function(){
    var result = util.Request.getUrlParam($('#url').val());
    $('#urlParam').val(result);
});
            
util.Request.getUrlParam()
该函数功能是将指定参数的值从URL提取出来；入参需要字符匹配，返回当前URL中对应参数的值。
注：当手动在浏览器地址栏添加参数时，需要回车输入URL

$('#format').on('click', function(){
    var date = new Date($('#date').val());
    var resultDate = util.DateUtil.format(date, $('#pattern').val());
    $('#resultDate').val(resultDate);
});

util.DateUtil.format(date, pattern)
该函数功能是将Date类型的数据按照pattern提供的格式转换为，一样格式的字符串
第一个入参是一个Date数据(在页面输入时，请尽量按照常用的时间格式来写)，第二个入参是格式字符串

util其他方法说明

源码 说明 
StringUtil.StrZeroAdd(12421); = "21"
StringUtil.StrZeroAdd("jqwgrqw"); = "qw"
            
对数字或字符串截取最后两位
            
StringUtil.isBlank(' ') = true;
StringUtil.isBlank('') = true;
StringUtil.isBlank(' A') = false;
StringUtil.isBlank('AAAA') = false;
            
判读字符串是否为空
            
$.ajax(url,Util.ObjectUtil.convertMethodToUndefined(param));
            
使用当处理方法的数据不能带有Function时使用。
在有些情况中（$.ajax(url,param)），param入参的属性中会带有
function类型的值，这里可以将其中的function过滤掉
